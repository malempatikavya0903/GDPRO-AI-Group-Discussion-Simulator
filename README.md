## Overview
GDPRO is an AI-powered group discussion simulator built using Gemini AI Studio.
It simulates real-time group discussions with multiple AI personas for placement training and debate practice.

## Problem Statement
Students often lack access to realistic group discussion environments with diverse viewpoints and structured feedback.

## Solution
GDPRO uses Gemini-based AI agents to:
- Simulate multiple panelists
- Present arguments and counter-arguments
- Maintain structured discussion flow

## Tech Stack
- Gemini AI Studio
- Gemini 3 / Gemini Pro
- Prompt orchestration & multi-agent logic
